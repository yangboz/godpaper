Packagefeathers.controls
Classpublic class TextInput
InheritanceTextInput FeathersControl Sprite DisplayObjectContainer DisplayObject EventDispatcher Object
Implements IFocusDisplayObject

A text entry control that allows users to enter and edit a single line of uniformly-formatted text.

To set things like font properties, the ability to display as password, and character restrictions, use the textEditorProperties to pass values to the ITextEditor instance.

See also

http://wiki.starling-framework.org/feathers/text-input
